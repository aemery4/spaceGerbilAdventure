Setup Guide

Do these steps once — then you're set forever!

1
Open VS Code

Find VS Code on your computer and open it. It's the blue icon that looks like angle brackets < >

2
Install Live Server

This lets you open the game in your browser with one click. You only install it once!

  • ➤ On the left side of VS Code, click the Extensions icon — it looks like 4 squares (⊞)
  • ➤ In the search box at the top, type: Live Server
  • ➤ Find the one that says "Live Server" by Ritwick Dey
  • ➤ Click the blue Install button
  • ➤ Wait a few seconds — done! ✅
💡 Tip: If you see "Uninstall" instead of "Install", you already have it — skip to Step 3!
3
Open the Game Folder

Tell VS Code where the game files live.

  • ➤ In VS Code, click File in the top menu
  • ➤ Click Open Folder...
  • ➤ Find the game_modules folder (wherever you unzipped it)
  • ➤ Click Select Folder

You should now see these files on the left:

📁 js
📄 index.html  ← the game!
📄 CLAUDE.md
📄 SETUP.html
4
Launch the Game!

Almost there! Here's how to open the game every time:

  • ➤ In the file list on the left, click on index.html
  • ➤ Right-click on index.html
  • ➤ Click "Open with Live Server"
  • ➤ Your browser opens with the game! 🎮
💡 From now on: just right-click index.html → Open with Live Server. That's the whole thing!
